# devcontainer Templates

## How to Use

when you want to use Python

```sh
curl -fsSL https://raw.githubusercontent.com/WangeSpyAI/devcontainer-templates/main/init.sh | bash -s python
```

when you want to use React

```sh
curl -fsSL https://raw.githubusercontent.com/WangeSpyAI/devcontainer-templates/main/init.sh | bash -s react
```
